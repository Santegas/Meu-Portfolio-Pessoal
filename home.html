Olá sou Santiago Oliveira!

Sou o Santiago Oliveira, um desenvolvedor que acredita que a tecnologia tem o poder de simplificar e transformar. Minha paixão é tirar ideias do papel e construir soluções digitais que não apenas funcionem, mas que sejam sólidas, seguras e eficientes no mundo real.

Ao longo da minha jornada, concentrei-me em dominar o ciclo completo de desenvolvimento, desde a arquitetura de banco de dados até a experiência final do usuário. Sou orientado a resultados e estou sempre em busca do aprendizado contínuo, focado em entregar a melhor solução para cada desafio.

Sinta-se à vontade para explorar meus projetos e entender como posso agregar valor ao seu próximo desafio.

Minha experiência me permite transitar da lógica robusta do back-end à interface intuitiva do front-end, sempre com o compromisso de construir aplicações que sejam tão confiáveis quanto inovadoras.

Minhas experiências

Zentris

Aqui ajudei a desenvolver soluções personalizadas, robustas e escaláveis que automatizam processos, integram dados e impulsionam a eficiência do negócio.

Zentris Atende, uma plataforma de gestão de confirmação de agendamentos para pequenas clínicas médicas com integração com IA e WhatsApp que participei da criação. Está sendo desenvolvido usando vue.js, html e css no front-end e java com spring no back-end. Nele temos authentication e authorization com JWT, dashboards, filtros, login page e uma tela de onboarding para clientes que ainda estamos implementando.

tecnologias mais usadas:

  • Vue.Js
  • Java +21
  • PostgreSQL

Meus projetos no github

task-manager-api

API de Gerenciamento de Tarefas (Task Manager API). Este projeto é uma API RESTful completa e funcional para gerenciar tarefas de forma eficiente. Desenvolvida em Java com Spring Boot, a API foi criada para demonstrar a aplicação de tecnologias modernas e boas práticas de desenvolvimento de software em uma solução de ponta a ponta.
